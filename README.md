# Pac-Man

一个使用 **Rust + Bevy** 构建的吃豆人小游戏。  
通过 ECS 架构实现清晰的模块分层、AI 行为、碰撞系统与动画逻辑，作为 Rust 游戏开发的学习项目。

## 项目简介

- **引擎**：Bevy 0.17.2
- **语言**：Rust 2024

## 快速开始

```bash
# 克隆项目
git clone https://github.com/iAeternus/pac-man.git
cd pac-man

# 构建并运行
cargo run
```

## 开发计划

### 核心系统

- [x] 游戏状态机（菜单 / 游戏中 / 暂停 / 结束状态切换）
- [x] 地图系统（静态地图加载与墙面碰撞检测）
- [ ] 玩家系统（键盘输入、移动、吃豆逻辑）
- [ ] 鬼系统（追逐、逃跑、巡逻行为）
- [ ] 道具系统（普通豆与能量豆逻辑）
- [ ] 碰撞系统（玩家与鬼、豆子碰撞检测）
- [ ] 分数系统（实时分数记录与展示）
- [ ] 输入系统（↑↓←→ / 空格 / 回车）

### 动画与表现

- [ ] 动画系统（吃豆人嘴巴开合、鬼闪烁）
- [ ] 粒子特效（吃豆与死亡时粒子反馈）
- [ ] 多关卡支持（JSON 地图加载与关卡切换）

### AI 与逻辑扩展

- [ ] AI 路径规划（A* 算法与不同鬼性格）
- [ ] 难度调整与速度曲线
- [ ] 鬼复活逻辑与基地返回

### 音效与调试

- [ ] 音效支持（吃豆、死亡音效）
- [ ] 调试工具（Bevy Inspector 支持）
